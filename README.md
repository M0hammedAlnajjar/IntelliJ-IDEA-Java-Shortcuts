# IntelliJ IDEA Java Shortcuts

A collection of useful **IntelliJ IDEA shortcuts for Java development** (Windows/Linux) to improve coding speed and productivity.

## General Shortcuts

| Shortcut               | Function                                           |
| ---------------------- | -------------------------------------------------- |
| `Shift` (twice)        | Search Everywhere                                  |
| `Ctrl + Space`         | Basic code completion                              |
| `Ctrl + Shift + Space` | Smart code completion                              |
| `Alt + Enter`          | Show quick fixes and suggestions                   |
| `Ctrl + Alt + L`       | Format code                                        |
| `Ctrl + /`             | Comment/uncomment a line                           |
| `Ctrl + Shift + /`     | Comment/uncomment a block                          |
| `Ctrl + D`             | Duplicate the current line                         |
| `Ctrl + Y`             | Delete the current line                            |
| `Ctrl + Shift + â†‘/â†“`   | Move a line up/down                                |
| `Ctrl + Alt + V`       | Extract variable                                   |
| `Ctrl + Alt + M`       | Extract method                                     |
| `Shift + F6`           | Rename variable, method, or class                  |
| `Ctrl + B`             | Go to declaration                                  |
| `Ctrl + Alt + B`       | Go to implementation                               |
| `Ctrl + F12`           | Show file structure                                |
| `Ctrl + F`             | Find                                               |
| `Ctrl + R`             | Replace                                            |
| `Ctrl + N`             | Find class                                         |
| `Ctrl + Shift + N`     | Find file                                          |
| `Alt + Insert`         | Generate code (constructor, getters/setters, etc.) |
| `Shift + F10`          | Run the project                                    |
| `Shift + F9`           | Debug the project                                  |

## Java Live Templates

Type the abbreviation and press **Tab** to generate code quickly.

| Template | Generates                                |
| -------- | ---------------------------------------- |
| `psvm`   | `public static void main(String[] args)` |
| `sout`   | `System.out.println();`                  |
| `fori`   | `for` loop with index                    |
| `iter`   | Enhanced `for` loop                      |
| `ifn`    | `if (variable == null)`                  |
| `inn`    | `if (variable != null)`                  |
| `try`    | `try-catch` block                        |

* Improve productivity
